# cucumber-x
Cucumber JS Extra - Simple IOC, REST Assertions etc.,

```sh
npm install cucumber-x --save
yarn add cucumber-x
```
 
## Usage

### Javascript

```javascript
var cucumberx = require('cucumber-x');
```

```sh
Output should be 'Boys'
```
### TypeScript
```typescript
import { cucumberx } from 'cucumber-x';

```
